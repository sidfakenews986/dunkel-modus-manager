# Dunkel Modus Manager

Willkommen beim Dunkel Modus Manager! Dieses Projekt bietet Entwicklern und Designern eine einfache Möglichkeit, den Dunkelmodus für ihre Webseiten und Anwendungen zu implementieren und anzupassen.

## Funktionen
- **Einfache Aktivierung**: Schalte den Dunkelmodus mit nur einem Klick ein oder aus.
- **Benutzerdefinierte Themes**: Erstelle und speichere deine eigenen dunklen Farbpaletten.
- **Automatische Erkennung**: Erkenne die Systemeinstellungen des Benutzers und passe den Dunkelmodus automatisch an.
- **Support für mehrere Plattformen**: Kompatibel mit Desktop- und Mobilanwendungen.

## Installation
1. Klone das Repository: `git clone https://github.com/deinbenutzername/dunkel-modus-manager`
2. Navigiere in das Projektverzeichnis: `cd dunkel-modus-manager`
3. Installiere die Abhängigkeiten: `npm install`

## Verwendung
Füge die Beispielcodesnippets in deine Webseite oder Anwendung ein, um den Dunkelmodus zu aktivieren:

```javascript
import { DarkModeManager } from 'dunkel-modus-manager';

const dmm = new DarkModeManager();
dmm.init();
```
